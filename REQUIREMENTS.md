# Требования к новому ticker-проекту

## Общая идея

Отдельный сервис для бегущих строк по видам спорта.

Первый набор:
- Tennis — готовая строка из старого проекта, два размера, можно перенести позже.
- Football / World Cup — новая строка, один размер.

## Главное отличие от match overlay

Это не оверлей конкретного матча.
Это постоянная информационная строка для эфиров.

Она должна уметь:
- брать новости;
- чистить/нормализовать их;
- показывать в едином визуальном стиле;
- управляться из Telegram.

## Приоритет MVP

1. Football ticker HTML.
2. API футбольных новостей.
3. Manual news через API.
4. Telegram bot для Football.
5. Подготовка к переносу Tennis.
