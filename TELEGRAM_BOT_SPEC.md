# Telegram bot spec

Лучше отдельный Telegram bot только под бегущие строки.

## Главное меню

```text
Бегущие строки

🎾 Tennis
⚽ Football / World Cup
```

## Football menu

```text
⚽ Football / World Cup ticker

Статус: включена
Режим: mixed
Новостей: 12
Скорость: 70

🔗 Ссылка для vMix
👀 Preview
➕ Добавить новость
🗑 Удалить новость
🧹 Очистить ручные новости
🔄 Обновить auto news
⚙️ Настройки
⬅️ Назад
```

## Режимы

- `auto` — только новости из источников;
- `manual` — только ручные новости;
- `mixed` — сначала ручные/закреплённые, потом auto.

## Manual add

Пользователь отправляет текст.
Бот сохраняет:

```json
{
  "source": "manual",
  "approved": true,
  "priority": 10,
  "title": "..."
}
```

## Safety

Бот не должен публиковать raw HTML.
Все новости экранировать.
Длинные новости резать до 140–180 символов.
